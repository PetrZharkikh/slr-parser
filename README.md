SLR(1) парсер для арифметических выражений с операциями `+ - * /` и скобками.  
Лексический анализ реализован с помощью Flex, сборка через CMake.

Грамматика:
```

S' -> E
E  -> E + T | E - T | T
T  -> T * F | T / F | F
F  -> ( E ) | id | num

````

## Сборка
```bash
cmake -S . -B build
cmake --build build -j
````

## Запуск

Программа читает выражение из стандартного ввода и печатает трассу работы SLR-парсера
(стек, вход и выполняемое действие).

```bash
echo "(a+3)*b" | ./build/slr_parser
```

В конце выводится `accept` или `error`.

## Тесты

В проекте есть отдельный тестовый бинарник.

```bash
./build/slr_tests
```

Примеры корректных выражений:

```
a+3
(a+3)*b
a+b*c
```

Примеры некорректных:

```
a+
(a+3
a**b
```
